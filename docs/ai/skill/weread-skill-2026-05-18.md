你好，我是悟空。

## 一、微信读书 Skill 是什么？

我是微信读书 APP 的重度使用者，最近微信读书推出了官方 Skill，真的让 AI 成为了我的阅读搭档。

![](http://cdn.passjava.cn/uPic/image-20260518185224060rfKtUW.png)

这款 Skill 支持哪些功能？

- 查阅书架：浏览你的个人书架，快速了解藏书全貌
- 阅读统计：时长、天数、偏好深度分析，量化你的阅读习惯
- 笔记和划线：查看个人划线和想法，导出笔记，回顾阅读中的思考
- 书籍搜索：在书城搜索任意书籍，快速获取书名、作者、评分等关键信息
- 书籍详情：查看书籍详情、章节目录、阅读进度，了解你的阅读旅程
- 推荐好书：基于你的阅读偏好，个性化推荐或相似书籍推荐

安装步骤在文末，还是非常简单的。

## 安装微信读书 Skill

官方地址：https://weread.qq.com/r/weread-skills

复制指令，丢给 AI Agent 安装，或者把压缩包下载下来，手动安装下，我是用 CC-Switch 手动安装的。

然后把 API key 丢给 AI Agent 自动配置下，就可以用这款 Skill 了。

![](http://cdn.passjava.cn/uPic/image-20260518193826475zLCqpf.png)

## 总结

这款 Agent Skill 能根据我的阅读习惯，越来越懂我，**真正实现了"千人千面"的阅读体验**。过去找书、查笔记、看进度需要在 APP 里反复切换页面，现在一句话就能搞定，**阅读管理的效率提升了一个量级**。更重要的是，它把散落在各处的阅读数据串联了起来——书架、笔记、时长、偏好不再是孤岛，而是形成了一个完整的个人阅读知识库，随时待命、随问随答。
