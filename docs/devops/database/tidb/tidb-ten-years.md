大家好，我是【悟空】，是 悟空聊架构 公众号的主理人。

很开心见证 TiDB 下一个十周年的开启，衷心祝愿 TiDB 发展越来越好

我在【 2025 年份】深入接触 TiDB，从参加 TiDB 社区征文开始，获得了四大奖项，同时获得了 TiDB 社区 2025 年度 MVA。

TiDB 作为国产分布式数据库的领跑者，在国际舞台上也占据重要席位，更在国内积累了深厚的社区土壤。近年来，我亲眼见证了它从产品到生态的持续进化。

真心希望更多同行能认识 TiDB、用上 TiDB——无论是面对数据伸缩、多模存储还是高可用挑战，它都可能成为你手中那个优雅的解决方案。

### 问题1：您会推荐大家在哪些场景下使用 TiDB？

- 原生分布式与弹性扩展场景。
- HTAP 混合负载场景。
- 高可用与多活容灾场景。
- 云原生与多云部署场景。
- 替换传统中间件与数据库瓶颈场景。

### 问题2：TiDB 给您留了下哪些深刻的印象？

- “真正的”一体化 HTAP。

- 极致的弹性与可扩展性。
- 强大的开源社区与生态。
- 持续的前瞻性产品演进。

### 问题3：数据库从业者了解 TiDB 有哪些好处？

- 掌握未来架构技能。
- 提升解决问题能力。
- 拥抱活跃的开源社区。
- 拓宽职业发展路径。

### 问题4：给正在选型数据库的朋友的建议

- 选型没有“银弹”，必须围绕真实的业务需求。
- 在扩展性、一致性与复杂度间做选择。
- 长远眼光：考量总体拥有成本与平滑演进。
- 充分进行概念验证。

### 问题5：对平凯数据库新品的期待

- 与 AI 结合，更高效地支持向量数据。
- 智能自治运维。
- 多模数据融合。
