在Linux系统中，可以使用`tar`命令来解压`.tar.gz`文件。以下是具体的步骤和命令：

### 基本命令格式
```bash
tar -xzf 文件名.tar.gz -C 解压目标目录
```

### 参数说明
- `-x`：表示解压文件。
- `-z`：表示文件是经过gzip压缩的。
- `-f`：指定文件名。
- `-C`：指定解压到的目标目录（可选，如果不指定，默认解压到当前目录）。

### 示例
假设你有一个名为`example.tar.gz`的文件，想要将其解压到当前目录下的`output`文件夹中，可以执行以下命令：
```bash
tar -xzf example.tar.gz -C output
```

如果不想指定解压目录，直接解压到当前目录，可以省略`-C`参数：
```bash
tar -xzf example.tar.gz
```

### 注意事项
1. 确保你有足够的权限来解压文件。如果没有权限，可以使用`sudo`命令提升权限：
   ```bash
   sudo tar -xzf 文件名.tar.gz -C 目标目录
   ```
2. 如果文件名中包含空格或特殊字符，需要用引号将文件名括起来，例如：
   ```bash
   tar -xzf "文件名.tar.gz" -C 目标目录
   ```
