## 部署网站

这次我将这个页面部署到我的网站上。

先将文件丢到服务器上，然后加一个 nginx 配置文件，接着配置下 nginx。

```JSON
server {
    listen       80;
    server_name  xx.xx.xx;

    location / {
        root   /home/ubuntu/jay/;
        index  simple.html;
    }
}
```

配置完成后，别忘了重启 nginx 服务。

```
sudo nginx -s reload
```

然后配置下 DNS 域名解析
