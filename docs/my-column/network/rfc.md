## RFC

网络协议就是这样，是一种“方言”，互相要用对方听得懂的方式对话。如果语法出现了问题，我们的自然语言就是“不明白你的意思，你说啥”。在 HTTP 这个“方言”里，就是用 HTTP 400 表达了同样的意思。

### HTTP

熟悉 HTTP 协议的方法，就是熟读 RFC2616，以及 2014 年 6 月的更新 RFC（7230, 7231, 7232, 7233, 7234, 7235）。

```
Hypertext Transfer Protocol -- HTTP/1.1
https://datatracker.ietf.org/doc/html/rfc2616#section-14.8
```
