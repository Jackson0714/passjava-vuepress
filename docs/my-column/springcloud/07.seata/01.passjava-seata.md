## 官方文档

https://seata.apache.org/zh-cn/docs/user/quickstart

## 下载 Seata

https://seata.apache.org/unversioned/download/seata-server/

![](http://cdn.jayh.club/uPic/image-20241115224258386E53rWH.png)

## 配置 Seata

将 Seata 服务注册到 nacos，需要修改注册中心的配置，有一个 application.example.yml 文件可以参考。

![](http://cdn.jayh.club/uPic/image-2024111522415280502w0XO.png)

## 运行 Seata 服务

``` SH
sh seata-server.sh -p 8091 -h 127.0.0.1 -m file
```

![](http://cdn.jayh.club/uPic/image-202411152244049365QjA35.png)

停止 Seata 服务

``` SH
ps -ef | grep seata                            

kill <seata 进程 id>
```
